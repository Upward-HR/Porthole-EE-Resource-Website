# Porthole Hospitality Employee Portal

This workspace contains a static employee resource portal for Porthole Hospitality.

## Files

- `index.html` — the employee portal home page

## How to use

1. Open `index.html` in a browser.
2. Review and update the document links, HR contacts, and onboarding details.
3. If you want to add more pages, create new HTML files and link them from `index.html`.

## Next steps

- Add a navigation menu for multiple sections
- Extract CSS to `styles.css` for easier editing
- Add search or a document library
- Convert to a React/Next.js app if you want dynamic features

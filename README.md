# Porthole Hospitality Employee Portal

A static, single-file HTML employee resource portal for **Porthole Hospitality**, a restaurant located at 20 Custom House Wharf, Portland, ME 04101.

## Overview

The Employee Resource Center serves as an internal-only landing page for new and existing employees, centralizing access to:
- Company handbook and HR policies
- Benefits information and documentation
- Emergency procedures and workplace safety protocols
- Key contacts and operational reminders

## Technology

- **Framework**: None - Pure HTML & CSS
- **Build system**: None required
- **Dependencies**: 
  - Google Fonts: Playfair Display, Source Sans 3

## Files

- `porthole-employee-portal.html` - Main portal (single-file application)
- `claude.md` - Project documentation and design system

## Getting Started

Simply open `porthole-employee-portal.html` in any modern web browser.

## Features

✅ Responsive design (desktop & mobile)  
✅ Company handbook links  
✅ HR contact information  
✅ Benefits guide and EAP details  
✅ Emergency action plan links  
✅ Workplace injury protocols  
✅ Quick employee reminders (pay schedule, time-off, discounts)
